# Course Syllabus: Learn Dev Tools Basics

This syllabus provides a summary of each lesson in the course. Lessons are organized by branch and build upon each other sequentially.

---

## 01-Install-Mise

**Problem:** Managing multiple development tools (Python, Node.js, etc.) traditionally requires learning separate version managers for each, creating unnecessary complexity.

**Hands-on:** Install mise in a GitHub Codespace using `curl https://mise.run | sh`, configure shell activation in `~/.bashrc`, and verify the installation works.

**Outcome:** You can use mise as a unified tool manager, ready to install and switch between any dev tools with a single consistent interface.
